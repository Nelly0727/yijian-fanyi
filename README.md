# 一键翻译 · GitHub 划词翻译插件

> 专为不太熟悉英文的开发者设计，在 GitHub 上划选文字即可翻译，一键翻译整个页面，还能解除复制限制。

-----

## ✨ 功能

- **划词翻译**：选中任意英文文字，气泡弹出中文翻译
- **全页翻译**：点击右下角按钮，整页英文一键换成中文，再点还原
- **解除复制限制**：GitHub 上复制不了的文字，装了就能复制
- **技术术语友好**：专业词汇保留英文并附中文解释，例如 fork（复刻）

-----

## 🚀 安装方法

> 目前需要手动安装，不需要任何编程基础。

**第一步：下载文件**

点击右上角绿色 `Code` 按钮 → `Download ZIP` → 解压

**第二步：打开 Chrome 扩展页面**

在地址栏输入：

```
chrome://extensions
```

**第三步：开启开发者模式**

右上角打开「开发者模式」开关

**第四步：加载插件**

点击「加载已解压的扩展程序」→ 选择解压后的文件夹

**第五步：填入 API Key**

点击工具栏插件图标 → 填入你的 [DeepSeek API Key](https://platform.deepseek.com/) → 保存

-----

## 🔑 关于 API Key

本插件使用 [DeepSeek](https://platform.deepseek.com/) 提供翻译，需要你自己申请 API Key。

- 注册地址：https://platform.deepseek.com/
- 费用极低，翻译几万字也花不了几分钱
- API Key 只保存在你自己的浏览器本地，不会上传到任何服务器

-----

## 📖 使用方法

|功能  |操作                        |
|----|--------------------------|
|划词翻译|在 GitHub 页面选中英文文字，自动弹出翻译气泡|
|复制原文|气泡内点「复制原文」                |
|复制译文|气泡内点「复制译文」                |
|全页翻译|点击页面右下角蓝色「译」按钮            |
|还原英文|再次点击右下角绿色「原」按钮            |
